<Update label="October 2024" tags={["New releases"]}>
## Human-in-the-Loop Workflows

New capability to pause agents for manual review on high-stakes decisions.

### Key Features
- **Conditional pausing** – Set rules like "if amount > $50,000, flag for review"
- **Notification system** – Get alerted via email or Slack when review is needed
- **Review interface** – Approve, reject, or modify agent decisions inline
- **Audit trail** – Track who approved what and when

### Use Cases
- Large quote approvals
- Contract signing authority
- Financial transaction verification
- Data quality checks

Example workflow:
```
If the quote total exceeds $50,000:
1. Mark status as "Needs Review"
2. Send notification to @sales-manager
3. Wait for human approval before sending
```
</Update>

<Update label="August 2024" tags={["New releases"]}>
## Event-Based Triggers

Automate workflows in real-time with event-based triggers.

### Trigger Types
- **Webhook triggers** – Run agents when external events occur
- **Email triggers** – Process new emails automatically
- **Worksheet changes** – React when rows are added or updated
- **Integration events** – Trigger on CRM updates, form submissions, file uploads

### Examples
- When invoice email arrives → Extract data → Update spreadsheet
- When form is submitted → Enrich lead → Add to CRM
- When file uploaded to Google Drive → Process document → Generate report

### Configuration
```
Trigger: New row in "Quote Requests" sheet
Condition: Status = "Pending"
Action: Run Quote Generation Agent
```

See our [Agent Guide](/guides/agents#setting-up-triggers) for details.
</Update>

<Update label="July 2024" tags={["Improvements"]}>
## API & Developer Tools

Expanded API capabilities for programmatic access.

### New Endpoints
- **Agents API** – Create, update, list, and delete agents
- **Runs API** – Trigger runs, check status, retrieve results
- **Worksheets API** – Read and write worksheet data
- **Sources API** – Upload and manage source documents

### Developer Experience
- **API documentation** – Interactive API reference with examples
- **SDKs** – Python and Node.js client libraries (coming soon)
- **Webhooks** – Get notified when runs complete or fail
- **Rate limits** – 1,000 requests/hour (Professional), 10,000/hour (Enterprise)

### Example
```bash
curl -X POST https://api.decisional.com/v1/agents/{agent_id}/run \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"input_data": {...}}'
```

Visit our [API Reference](/api-reference/introduction) for full documentation.
</Update>
