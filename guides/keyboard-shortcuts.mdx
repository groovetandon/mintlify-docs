## Overview

Decisional supports keyboard shortcuts and slash commands to help you navigate and take actions quickly without reaching for your mouse.

## Slash Commands

Type `/` in the chat or command bar to access quick actions:

| Command | Description |
|---------|-------------|
| `/run` | Execute the current agent |
| `/test` | Run a test execution |
| `/build` | Rebuild the agent workflow |
| `/connect` | Open the integrations panel |
| `/settings` | Open agent settings |
| `/help` | Show available commands |

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + /` | Show keyboard shortcuts |
| `Cmd/Ctrl + S` | Save current changes |
| `Escape` | Close modal or panel |

## Agent Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Send message to agent |
| `Cmd/Ctrl + Shift + Enter` | Run agent |
| `Tab` | Navigate between panels |

## Spreadsheet Shortcuts

When working in the worksheet:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + C` | Copy selection |
| `Cmd/Ctrl + V` | Paste |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Arrow keys` | Navigate cells |
| `Enter` | Edit cell / Move down |
| `Tab` | Move to next cell |

<Tip>
  Press `Cmd/Ctrl + /` at any time to see all available shortcuts for your current context.
</Tip>

## Command Palette

The command palette (`Cmd/Ctrl + K`) provides quick access to:

- Navigate to any agent
- Switch workspaces
- Access settings
- Run common actions
- Search documentation

Start typing to filter available commands and press `Enter` to execute.
